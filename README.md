# ML Projects

A portfolio of machine learning notebooks covering regression, classification, clustering, tree-based models, Kaggle workflows, and neural networks.

![Portfolio overview](assets/readme/portfolio-overview.svg)

## Project Index

| Project | Focus | Main idea |
| --- | --- | --- |
| [Linear Regression](Linear_Regression/02-Linear%20Regression%20Project.ipynb) | Regression | Predict ecommerce customer spending from behavioral features. |
| [Logistic Regression](Logistic_Regression/02-Logistic%20Regression%20Project.ipynb) | Classification | Model ad-click behavior from advertising user data. |
| [KNN](KNN/02-K%20Nearest%20Neighbors%20Project.ipynb) | Classification | Apply distance-based learning on a structured feature dataset. |
| [SVM](SVM/02-Support%20Vector%20Machines%20Project.ipynb) | Classification | Separate Iris species with support vector machines. |
| [Decision Trees & Random Forest](dtree_and_rfc/02-Decision%20Trees%20and%20Random%20Forest%20Project.ipynb) | Classification | Compare tree-based models on LendingClub loan data. |
| [K-Means](Kmeans/kmeans_nour.ipynb) | Clustering | Implement and visualize K-means clustering from scratch. |
| [Titanic Kaggle](Titanic-kaggle/titanic_v3.0.ipynb) | Kaggle classification | Engineer features and predict Titanic passenger survival. |
| [LendingClub Neural Network](artificial_neural_network/LendingClub_project.ipynb) | Deep learning | Predict whether a borrower is likely to repay a loan. |
